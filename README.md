# mr.builder
a tool to ceate vagrant and virtualbox images

mr.builder is a wrapper arround $PACKER to create $VAGRANT and $VBOX images
easy and fast.

## Features

- creates vagarnt images
- create virtualbox image
- same config for both
- beautiful docs
